Brand Identity and Visual Design Project

This project involved creating a modern and memorable brand identity for a lifestyle startup. From initial concept sketches to the final logo and color system, the design focuses on clarity, balance, and a strong visual presence. The final deliverables included a complete brand style guide, social media assets, and web-ready visuals aligned with the client’s vision.

Responsive Web Application for Task Management

Developed a full-stack web app that helps teams organize tasks and track progress in real time. Built with React, Node.js, and MongoDB, the platform features an intuitive interface, live updates, and secure user authentication. The project emphasizes performance, accessibility, and clean UI design.

Urban Portrait Series – Light, Mood & Character

A visual storytelling project exploring personality and emotion through urban portrait photography. Shot entirely with natural light, the series captures raw expressions and everyday beauty within city environments. Post-production focused on tone balance and preserving authentic skin tones.
